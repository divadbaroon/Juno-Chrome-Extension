Profiles

Used to customize the behavior of Juno, shaping its interactions based on specific users, technologies, and desired persona traits.

Profile Overview

Attribute Description
name The name of the entity
gender Gender of the entity
language Language entity speaks in (Refer to documentation for available languages)
personality Describes the overall temperament of the entity
persona The entity will act as if they are this persona
prompt Prompt used to query GPT
role Role of the entity
gpt_model Model used for generating responses (Fine-tuning recommended. See /training)
package Optional package for added functionalities. See packages for more information
startup_sound Whether to play a startup sound
voice_name Voice used for text-to-speech. In this example, I am using a custom-made voice modeled after Barack Obama, created using Elevenlabs
text_to_speech_engine Engine used for text-to-speech (e.g., Elevenlabs or Azure)
voice_recognition_engine Engine used for voice recognition
user_name Name of the user interacting with the entity
user_gender Gender of the user
user_age Age of the user
