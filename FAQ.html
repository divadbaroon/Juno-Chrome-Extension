About Profile Packages FAQ

What is it?

Juno is best described as a platform that congregates the most advanced publicly available AI solutions to create a well-rounded assistant. It currently uses Azure for speech and intent recognition, GPT for dynamic language interaction, and Elevenlabs, OpenAI, or Azure for text-to-speech.

How do I use it?

Select a profile and package. Profiles customize how Juno behaves and packages dictate what functions it can execute. See more on the Profiles and Packages respective pages.

Can I create my own profiles and packages?

Yes, creating your own profiles and packages is supported and highly encouraged. See more on the Profiles and Packages respective pages for more information.

Can I use Juno in my projects?

Yes, Juno was created with ease of incorporation in mind. See the Usage section located within Juno's documentation, or visit the Code table for more details.
